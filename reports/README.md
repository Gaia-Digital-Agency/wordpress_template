# Project Reports

This directory contains generated markdown reports for your GitHub projects.

## Generated Reports

Reports are automatically generated with timestamps and saved here when you run the reporting commands.

## Report Types

### Single Project Report
Contains detailed information about one project including:
- Progress summary with completion percentages
- Status breakdown (Backlog, To Do, In Progress, Done, Blocked)
- Phase breakdown with task lists
- Priority breakdown
- Developer assignments

### All Projects Portfolio Report
Contains comprehensive overview of all projects including:
- Executive summary with portfolio-wide statistics
- Active projects overview table
- Detailed breakdown for each project
- Closed projects summary
- Portfolio health indicators
- Risk assessment

## Files

Reports are named with the following patterns:
- `project_{number}_{title}_{date}.md` - Single project reports
- `all_projects_portfolio_{date}.md` - All projects reports

All generated `.md` files are ignored by git to keep your repository clean.
